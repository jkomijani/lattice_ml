# lattice_ml

`lattice_ml` is a PyTorch-based library designed for lattice field theory.
It integrates key functionalities from various upstream repositories and is
structured to grow with new modules and features over time.

## Upstream Sources

This project (`lattice_ml`) integrates functionality from two separate upstream repositories:

- `lattice_ml.linalg` is based on [`torch_linalg_ext`](https://github.com/jkomijani/torch_linalg_ext)
- `lattice_ml.integrate` is based on [`torch_solve_ext`](https://github.com/jkomijani/torch_solve_ext)

We intentionally started this repository with a clean history.
